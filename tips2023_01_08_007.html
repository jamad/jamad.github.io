Q

    2023-01-08: how to join multiple movie files into a single file?

A

   the following code make file_c as file_a + file_b 


 >set file_a="C:\Users\jamad\output_ufo3.mp4"
>set file_b="D:\miscFiles\UFOpart5.mp4"
>set file_c="C:\Users\jamad\output_ufo4.mp4"
>ffmpeg -i %file_a% -i %file_b% -filter_complex "[0:v]scale=640x360,setsar=1[v0];[1:v]scale=640x360,setsar=1[v1];[v0][0:a][v1][1:a]concat=n=2:v=1:a=1[outv][outa]" -map "[outv]" -map "[outa]" %file_c%
