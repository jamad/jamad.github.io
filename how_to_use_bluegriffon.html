How to add a new page to my github pages (w/ Bluegriffon+ Github Desktop)


Bluegriffon

    1. create a new page (ctrl+T)

    2. save the page (ctrl+S)

    3. insert the link to the existing page (RMB context menu)

    4. save all updates


Github Desktop ( https://github.com/jamad/jamad.github.io )

    5. commit and push the files


Browser

    6. check the pages on (https://jamad.github.io)
