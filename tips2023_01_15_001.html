Q

    2023-01-15: (atCoder) how to simulate input() to check the code?

A

    see the following solution!


def solve():
    N=int(input())
    for i in range(N):
        s,t=input().split()
        print(s,t)

input_text='''\
5
a b
y z
c d
x y
b c
'''
   
import unittest.mock as mock

input_iter = iter(input_text.splitlines())

with mock.patch('builtins.input', lambda: next(input_iter)):
    solve()
