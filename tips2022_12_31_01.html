Question

    How to add javascript showing today's date? 

Answer

    use Blue Griffon >> Insert >> HTML markup >> copy following code

<span id="date"></span>
<script>
  window.onload = function() {
    var today = new Date();
    var options = { year: 'numeric', month: 'numeric', day: 'numeric' };
    document.getElementById("date").innerHTML = today.toLocaleDateString('en-US', options);
  }
</script> :
