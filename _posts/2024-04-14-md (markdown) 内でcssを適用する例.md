`<style>`タグを入れて、センタリングしてみるテスト。<br>
jekyll でhtml化された時だけ見た目に影響する。<br>
`place-items: center;` は外してみた。<br>
formatが崩れるので`*`は使わない


```html
<style>
body { 
  background: #232;
  text-align: center;   
}
</style>
```

<style>
body { 
  background: #232;
  text-align: center;   
}
</style>


# これは`#`の行
## これは`##`の行
