# 四字熟語辞書

JSON形式で拡張可能な四字熟語辞書アプリケーションです。

## ファイル構成

```
四字熟語辞書/
├── 四字熟語辞書_JSON版.html    # メインのHTMLファイル
├── idioms.json                # 四字熟語データベース
└── README.md                  # このファイル
```

## 使い方

1. `四字熟語辞書_JSON版.html` をブラウザで開く
2. 検索ボックスで四字熟語を検索
3. カテゴリボタンで行ごとにフィルタ

## データの追加・編集方法

### JSONファイルの構造

`idioms.json` は以下の形式で四字熟語を保存しています：

```json
[
  {
    "word": "一期一会",
    "reading": "いちごいちえ",
    "meaning": "一生に一度だけの出会い。その機会を大切にすること。"
  },
  {
    "word": "温故知新",
    "reading": "おんこちしん",
    "meaning": "古いことを研究して新しい知識を得ること。"
  }
]
```

### データの追加手順

1. `idioms.json` をテキストエディタで開く
2. 最後の項目の後にカンマ `,` を追加
3. 新しい四字熟語を同じ形式で追加：

```json
  ,
  {
    "word": "新しい四字熟語",
    "reading": "あたらしいよじじゅくご",
    "meaning": "意味の説明"
  }
```

4. ファイルを保存
5. ブラウザをリロードして確認

### 注意点

- 各項目は `{ }` で囲む
- 項目の区切りはカンマ `,` を使用
- 最後の項目にはカンマを付けない
- 読み仮名は**ひらがな**で記入（カテゴリ分類に使用）
- JSONの構文エラーがあると読み込めません

### JSONの検証

JSONファイルが正しく記述されているか確認するには：
- https://jsonlint.com/ などのオンラインツールを使用
- または、ブラウザの開発者ツール（F12）でエラーメッセージを確認

## データの例

### 基本的な追加

```json
{
  "word": "七転八起",
  "reading": "しちてんはっき",
  "meaning": "何度失敗しても諦めないこと。"
}
```

### 複数追加

```json
[
  {
    "word": "一期一会",
    "reading": "いちごいちえ",
    "meaning": "一生に一度だけの出会い。その機会を大切にすること。"
  },
  {
    "word": "温故知新",
    "reading": "おんこちしん",
    "meaning": "古いことを研究して新しい知識を得ること。"
  },
  {
    "word": "七転八起",
    "reading": "しちてんはっき",
    "meaning": "何度失敗しても諦めないこと。"
  }
]
```

## 機能

- **検索機能**: 四字熟語、読み仮名、意味から検索
- **カテゴリフィルタ**: あ行〜わ行で分類
- **レスポンシブデザイン**: スマートフォンでも快適に使用可能
- **収録数表示**: 総数と表示中の数をリアルタイム表示

## 現在の収録数

約600語の四字熟語を収録

## 技術仕様

- HTML5 + CSS3 + Vanilla JavaScript
- 外部ライブラリ不要
- ローカルで動作（サーバー不要）

## トラブルシューティング

### データが表示されない場合

1. `idioms.json` が同じディレクトリにあることを確認
2. ブラウザの開発者ツール（F12）でエラーメッセージを確認
3. JSONの構文が正しいか確認

### カテゴリフィルタが機能しない場合

- 読み仮名が**ひらがな**で記入されているか確認
- 特殊な文字が含まれていないか確認
