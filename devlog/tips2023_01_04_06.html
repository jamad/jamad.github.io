Q

    (ffmpeg) how to convert movie file (ex.  .mkv, .flv ) into .mp4?  

A

  •  ffmpeg -i aaa.flv -vcodec copy -vcodec h264_nvenc aaa.mp4  (using GPU) 
    •  ffmpeg -i aaa.flv -vcodec copy aaa.mp4  (without GPU, CPU becomes almost 100% and slow to finish)
  •  for better quality, try the following ( reference : https://stackoverflow.com/questions/49378977/ffmpeg-webm-to-mp4-quality-loss )
    •  ffmpeg -i aaa.webm -crf 1 -c:v libx264 aaa.mp4
