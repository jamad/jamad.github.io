Q

     VSCodeでIllustratorの作業を自動化する方法は？

  • 公式サンプルあるんかい！
    • 例　C:\Program Files\Adobe\Adobe Illustrator 2023\Scripting\Sample Scripts\JavaScript\Working With Paths\Text on Path.jsx
      • 　
  • 重要なtips
    • console.log() はエラー`console is undefined`になる。　使うべきは、$.writeln("hello world");

A

    準備と確認

  1. Launch Adobe Illustrator
  2. VSCode
  3. Install extendscript debugger, then relaunch VSCode
  4. F5 then select Extend script (suggested)  >> debugging mode Launch >> Adobe Illustrator 2023
  5. make a 1-liner .jsx file   `alert("hello there")`
  6. test F5 then see "hello there" message from Adobe Illustrator
